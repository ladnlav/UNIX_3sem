##7.execbg

*Программа обрабатывает аргументы: команду и список параметров к ней. Должна запускать программу в фоновом режиме. При отсутствии перенаправления вывода — перенаправить вывод запускаемой команды в файл принудительно. При завершении работы терминала программа должна продолжить работать*

##8.overwrite

*Так как фильтры не могут писать и читать один и тот же файл одновременно, то предлагается написать программу, которая запускалась бы в пайпе с фильтром и сохраняла бы весь вывод фильтра во временный файл, после чего копировала бы временный файл на место старого. Временный файл должен быть удален. При прерывании процесса во время записи временного файла — удалить временный файл. Старый должен остаться в исходном варианте. Запретить прерывать процесс при перемещении информации из временного файла в исходный. Инод обрабатываемого файла должен не измениться.*
Пример строки запуска программы:
`sort <file | overwrite file`
